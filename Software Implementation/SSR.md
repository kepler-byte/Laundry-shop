## 2. Functional Requirements (ความต้องการด้านฟังก์ชัน)

###  2.3 การจัดการคำสั่งงาน (Order Management)
ส่วนนี้เน้นการทำงานที่จุดรับ-ส่งผ้า (หน้าร้าน) การรับออเดอร์ และการคิดเงิน

| Requirement ID | ชื่อฟังก์ชัน (Feature Name) | รายละเอียดความต้องการ (Description) | ผู้ใช้งาน (Actor) |
| :--- | :--- | :--- | :--- |
| **FR-2.3-01** | **สร้างคำสั่งงานใหม่** (Create Order) | ระบบต้องสามารถบันทึกข้อมูลการรับผ้า โดยระบุข้อมูลลูกค้า, ประเภทผ้า, จำนวนชิ้น/น้ำหนัก, และประเภทบริการ (ซักอบ, ซักรีด, รีดอย่างเดียว) ได้ | พนักงานหน้าร้าน |
| **FR-2.3-02** | **คำนวณราคาอัตโนมัติ** (Calculate Price) | ระบบต้องคำนวณราคารวมอัตโนมัติตามอัตราค่าบริการที่กำหนดไว้ (เช่น คิดตามน้ำหนัก หรือคิดรายชิ้น) รวมถึงรองรับการใส่ส่วนลด | ระบบ (System) |
| **FR-2.3-03** | **ออกใบรับงาน/ใบเสร็จ** (Generate Receipt) | ระบบต้องสามารถพิมพ์ใบรับงาน (มี QR Code หรือ Barcode อ้างอิง) เพื่อให้ลูกค้าเก็บไว้เป็นหลักฐาน และใบกำกับสำหรับติดที่ถุงผ้า | พนักงานหน้าร้าน |
| **FR-2.3-04** | **ค้นหาและติดตามสถานะ** (Search Order) | ระบบต้องสามารถค้นหาคำสั่งงานได้จาก หมายเลขใบงาน, เบอร์โทรศัพท์ลูกค้า เพื่อตรวจสอบสถานะปัจจุบัน (เช่น รอซัก, กำลังซัก, เสร็จสิ้น) | พนักงานหน้าร้าน |
| **FR-2.3-05** | **ยกเลิกหรือแก้ไขคำสั่งงาน** (Edit/Cancel Order) | ระบบต้องอนุญาตให้แก้ไขรายละเอียดหรือยกเลิกคำสั่งงานได้ เฉพาะในกรณีที่สถานะยังไม่เข้าสู่กระบวนการซัก | ผู้จัดการ/พนักงาน |

## 2.4 การจัดการกระบวนการซัก/รีด (Laundry & Ironing Process Management)
ส่วนนี้เน้นการทำงานหลังร้าน (Back-office) การอัปเดตสถานะงานจริง และการควบคุมคุณภาพ

| Requirement ID | ชื่อฟังก์ชัน (Feature Name) | รายละเอียดความต้องการ (Description) | ผู้ใช้งาน (Actor) |
| :--- | :--- | :--- | :--- |
| **FR-2.4-01** | **แสดงคิวงาน** (View Job Queue) | ระบบต้องแสดงรายการผ้าที่รอการจัดการ โดยเรียงลำดับตามความเร่งด่วน (Urgency) หรือเวลาที่รับงาน (First-In, First-Out) | พนักงานซักรีด |
| **FR-2.4-02** | **อัปเดตสถานะรายขั้นตอน** (Update Stage) | ระบบต้องให้พนักงานกดอัปเดตสถานะเมื่อจบแต่ละขั้นตอนได้ เช่น "กำลังซัก" -> "กำลังอบ" -> "รอรีด" -> "กำลังรีด" | พนักงานซักรีด |
| **FR-2.4-03** | **บันทึกความเสียหาย/ตำหนิ** (Log Defect) | ระบบต้องมีฟังก์ชันให้บันทึกกรณีพบผ้าเสียหายหรือคราบที่ซักไม่ออก ระหว่างกระบวนการซัก/รีด (พร้อมแนบรูปภาพถ้ามี) เพื่อแจ้งลูกค้าทราบ | พนักงานซักรีด |
| **FR-2.4-04** | **ระบุเครื่องที่ใช้งาน** (Assign Machine) | (Optional) ระบบสามารถระบุได้ว่าตะกร้านี้กำลังซักหรืออบที่เครื่องหมายเลขใด เพื่อการติดตาม | พนักงานซักรีด |
| **FR-2.4-05** | **แจ้งเตือนงานเสร็จสิ้น** (Mark as Completed) | เมื่อรีดและแพ็คเสร็จสิ้น ระบบต้องเปลี่ยนสถานะเป็น "พร้อมส่งมอบ" และส่งการแจ้งเตือน (SMS/Line/Email) ไปยังลูกค้าอัตโนมัติ | พนักงาน/ระบบ |
